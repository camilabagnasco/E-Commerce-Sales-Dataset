# Amazon Fashion Sales Analysis

## Project Overview

This project analyzes sales data from an Amazon fashion retailer to identify key revenue drivers, customer demand patterns, and operational improvement opportunities. The analysis combines data cleaning, exploratory data analysis, and business intelligence techniques to generate actionable insights for decision-making.

## Business Problem

E-commerce businesses generate large volumes of transactional data, but transforming that data into actionable insights can be challenging. This project aims to answer critical business questions related to sales performance, product demand, geographic distribution, and operational efficiency.

## Tools Used

* Python (Pandas, NumPy, Matplotlib)
* Kaggle Notebooks
* MySQL
* Tableau
* GitHub

## Methodology

1. Data Cleaning and Quality Assessment
2. Feature Engineering
3. Exploratory Data Analysis (EDA)
4. Business Insights Generation
5. Dashboard Development
6. Business Recommendations

## Key Findings

* Generated over $78.5 million in revenue from 113,004 unique orders.
* The "Sets" category was the highest-performing product segment.
* Maharashtra, Karnataka, and Telangana were the top revenue-generating states.
* An 8.9% order cancellation rate represents a potential opportunity for operational improvement.
* A small number of products accounted for a significant share of total units sold.

## Business Recommendations

* Expand inventory and marketing efforts for high-performing categories.
* Prioritize growth initiatives in top-performing geographic markets.
* Investigate drivers of order cancellations to improve conversion and fulfillment.
* Maintain adequate stock levels for top-selling products.
* Leverage product demand patterns to optimize promotional strategies.

## Deliverables

* Data Cleaning & EDA Notebook
* SQL Analysis
* Interactive Tableau Dashboard
* Business Insights & Recommendations

# Análisis de Ventas de Moda en Amazon

## Descripción del Proyecto

Este proyecto analiza datos de ventas de una empresa de moda que comercializa sus productos a través de Amazon. El objetivo es identificar los principales impulsores de ingresos, comprender los patrones de compra de los clientes y detectar oportunidades de mejora operativa mediante el uso de herramientas de análisis de datos y Business Intelligence.

## Problema de Negocio

Las empresas de e-commerce generan grandes volúmenes de datos transaccionales cada día. Sin embargo, convertir esos datos en información útil para la toma de decisiones puede resultar complejo. Este análisis busca responder preguntas clave relacionadas con rendimiento de ventas, categorías de productos más rentables, distribución geográfica de la demanda, productos con mayor volumen de ventas.

## Herramientas Utilizadas

* Python (Pandas, NumPy, Matplotlib)
* Kaggle Notebooks
* Tableau
* GitHub

## Metodología

1. Evaluación de la calidad de los datos.
2. Limpieza y transformación de datos.
3. Creación de nuevas variables (Feature Engineering).
4. Análisis Exploratorio de Datos (EDA).
5. Identificación de insights de negocio.
6. Elaboración de recomendaciones estratégicas.
7. Desarrollo de dashboards interactivos en Tableau.

## Principales Hallazgos
* Se generaron más de $78,5 millones en ingresos durante el período analizado.
* Se registraron 113.004 órdenes únicas.
* El valor promedio por pedido fue de ₹648,57.
* La categoría "Sets" fue la principal generadora de ingresos.
* Maharashtra, Karnataka y Telangana fueron los estados con mayores ventas.
* La tasa de cancelación alcanzó el 8,9%, representando una oportunidad de mejora operativa.
* Un número reducido de productos concentró una parte significativa de las ventas totales.

## Recomendaciones de Negocio

* Potenciar las categorías de mayor rendimiento
* Incrementar la disponibilidad de inventario y las acciones de marketing para las categorías con mayor generación de ingresos.
* Priorizar los mercados más rentables
* Concentrar campañas promocionales y estrategias comerciales en los estados que generan la mayor parte de los ingresos.
* Reducir las cancelaciones
* Investigar las causas principales de cancelación para minimizar la pérdida de ventas y mejorar la experiencia del cliente.
* Optimizar la gestión de inventario
* Garantizar niveles adecuados de stock para los productos con mayor demanda y rotación.
* Aprovechar patrones de demanda
* Utilizar los productos más vendidos como base para promociones cruzadas y estrategias de crecimiento.

## Entregables

* Notebook de limpieza y análisis exploratorio de datos.
* Consultas SQL para análisis de negocio.
* Dashboard interactivo desarrollado en Tableau.
* Insights y recomendaciones basadas en datos.
